Go Home Navi
Safe and Efficient Return Home Support

From checking last trains and taxi fares to searching for accommodations. We support your safe return home.

Download Now

Key Features

Return Home

Display the route search screen to your home with just a tap on the icon. With Quick Mode enabled, the route search screen opens automatically when you launch the app.

Check Last Train

Search for the last train using route search sites. If you register your home in advance, you can get last train information from your current location to your home with one tap.

Check Taxi Fare

Easily check taxi fares to your home or destination. You can quickly estimate taxi fares even if you miss the last train.

Book Accommodation

Integrated with Rakuten Travel, you can search and book nearby accommodations. Convenient when returning home is difficult.

Display Map

Display a map of your current location within the app. Useful for finding nearby places to spend time if you have no way to get home.

Notification Feature

Set your return time and receive notifications at the specified time. With Quick Mode enabled, tapping the notification opens the map app and displays the route search screen to your home.

Useful Scenarios

Office Worker Catching the Last Train

Quickly check the last train time while working overtime. Find the optimal route home and return without rushing. Use the app's notification feature to focus on work without forgetting the last train time.

Safe Return After a Night Out

Check taxi fares in advance after an enjoyable night out. Supports late-night rates, allowing you to prepare for your return home with peace of mind. Also displays nearby taxi stand information for a smooth return home.

Sudden Schedule Changes During Business Trips

Even if plans change and you can't make the last train, quickly book nearby accommodations. Flexibly respond to unexpected issues during business trips.
